---

## RULE #1 — NEVER GO BACK TO A FILLED FIELD
Only move forward. If a field has any value, skip it.

## RULE #2 — RECONNAISSANCE BEFORE FILLING
Scroll through the entire form once before filling anything.

## RULE #3 — READ THE QUESTION FIRST
Read the label before filling.

---

## UNKNOWN FIELDS — defaults
- "How did you hear about us?" → LinkedIn
- "Desired start date?" → Immediately / 2 weeks
- "Are you 18 or older?" → Yes
- "Preferred name" → {{FIRST_NAME}} | "Middle name" → blank

---

## DROPDOWN HANDLING

**Standard HTML select:** CLICK to open → READ all visible options → CLICK the best match.

**Custom typeahead:** CLICK field → TYPE 3-4 letters → WAIT 3 seconds → press ENTER.

---

## RADIO BUTTONS
Click the CIRCLE, not the text.

- "Authorized to work in US?" → YES
- "Require visa sponsorship?" → NO
- "US citizen?" → YES
- "Willing to relocate?" → NO

---

## CHECKBOXES
Click the SQUARE to check it. Once checked, do NOT uncheck.
